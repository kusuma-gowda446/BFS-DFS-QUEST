⚡ BFS (Breadth-First Search)

BFS operates on a First-In, First-Out (FIFO) policy using a Queue. It represents systematic exploration, expanding outwards like scouts in a wide plain.

  • All immediate neighbors (depth 1) are explored first, before moving to neighbors of neighbors (depth 2).
  • In-game, new neighbors are added to the back of the Scroll Queue.
  • You can only explore nodes connected to the Front of your queue.
  • BFS is perfect for finding the shortest path in an unweighted graph!

🔥 DFS (Depth-First Search)

DFS operates on a Last-In, First-Out (LIFO) policy using a Stack. It represents deep, daring exploration, plunging deep into dark dungeons first.

  • It plunges deep down a single path. Only when it hits a dead end does it backtrack to the nearest intersection.
  • In-game, nodes are placed on top of the Memory Crystal Stack.
  • You can only explore nodes connected to the Top active crystal.
  • If there are no unvisited neighbors, the game enters Backtracking Mode: the top crystal shatters and you return to the previous node to find alternative paths.
